1. Install GameDistribution .unitypackage: https://github.com/GameDistribution/gd-sdk-unity?tab=readme-ov-file

2. Create new assembly definition in the root of the imported GameDistributionSDK folder, called exactly 'GameDistributionSDK'.

3. Install GameDistribution API package:

```git
https://github.com/Prime-SDK/GameDistribution-API.git
```

4. Make sure you've selected WebGL in Build Settings.

5. Open Toolkit, choose GameDistributionConfiguration for Build.

6. For banners, use prefabs they provided in GameDistributionSDK package.

7. For payments, use XSolla API package for PrimeSDK (available in Toolkit).
